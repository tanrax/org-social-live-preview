Org Social Live Preview

Generate beautiful preview cards for Org Social posts in real-time.

Usage

Add a ?post= parameter with the URL of an Org Social post:

http://{{ domain }}{% if show_port %}:{{ port }}{% endif %}/?post=https://foo.org/social.org#2025-02-03T23:05:00+0100

URL Encoding

The post URL should be URL-encoded:

http://{{ domain }}{% if show_port %}:{{ port }}{% endif %}/?post=https%3A%2F%2Ffoo.org%2Fsocial.org%232025-02-03T23%3A05%3A00%2B0100

Unix command to encode URLs:

python3 -c "import urllib.parse; print(urllib.parse.quote('https://foo.org/social.org#2025-02-03T23:05:00+0100'))"
